Fátima Click
Description:
Fátima Click is a website designed to centralize information about religious life and Catholic events. It includes features for checking Mass schedules, daily prayers, and events related to the Catholic Church, with a special focus on the Fátima community. This project was developed using HTML, CSS (with Bootstrap), JavaScript, and Python.

Features:

Mass Schedules: Display of Mass schedules for various churches in Fátima.
Prayers: Availability of daily and special Catholic prayers.
Events: Calendar of Catholic events, such as gatherings, celebrations, and retreats.
Responsive Design: Layout optimized for various devices using the Bootstrap grid system.
Interactivity: Use of interactive components like modals, sliders, and dropdown menus to enhance navigation.
Technologies Used:

HTML5
CSS3 (Bootstrap)
JavaScript
Python (for additional backend functionalities or data handling)
Objective:
To make it easier for people to access information about religious life in Fátima and encourage active participation in events and prayers. The project focuses on simplicity and usability, providing a pleasant and practical experience.
